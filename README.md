# 拉博电器项目：LN905F （MCH+NTC自动校准）

# 自动校准操作说明
断开AC。按下电源键再插入AC电源。等待蜂鸣器滴一声且上面四个灯亮起松开按键开始进行自动校准。  
注意：自动校准会将MCH保持200度。如若校准结束时温度不在200度则NTC测量有偏差需要手动校准。  
#### 校准状态指示
1. 校准中：四个灯代表校准次数。慢闪时等待温度上升，快闪时代表等待温度稳定。  
2. 校准完成：从第一个灯开始点亮到最后四个灯亮然后熄灭，一直循环。  (循环流水灯)
3. 校准结束：四个灯全亮。  
注意：再校准完成时温度的正负偏差是3度，此时可以拔下电源完成校准。若继续等待到校准结束则温度正负偏差小于0.5度（建议等待设备校准结束后拔下电源）。  
#### 在自动校准结束后如果温度不在200度左右可手动操作进行修改校准值。(尽量保持正负3度以内)  
按下设定键上调温度。  
按下锁定键下调温度。  
注意：手动调整完温度后需要手动保存，按下开/关机键进行保存。按上下调节单次调节一度。  

# 测试说明
1 短路烧录引脚上电进入测试模式。亮起中间的灯且滴一声代表成功进入。  
2 按下设定键打开所有LED以及可控硅。  
3 按下锁定键关闭所有LED以及可控硅。  
4 按下电源键检测电源是否为220v/50Hz。  
5 完成滴滴三声，所有LED熄灭。  
6 如若电源出错则亮起LED2:电压错误/LED3：频率错误。且不发出响声。
